# iBuy Luxury Cars — редизайн

Полный редизайн [ibuylc.com](https://www.ibuylc.com/) — лидоген на выкуп люксовых
и экзотических машин, подразделение Marshall Goldman Motor Sales.

## Запуск

```bash
python3 04-build/serve.py        # → http://localhost:5173
python3 04-build/bundle.py       # → одностраничное превью в 03-design/
```

Технические детали, правила и открытые вопросы — в [04-build/README.md](04-build/README.md).

## Структура

| Папка | Что там |
|---|---|
| `00-brief` | заметки клиента + разбор |
| `01-reference/current-site` | аудит текущего сайта, скриншоты |
| `01-reference/inspiration` | референсы и их разбор |
| `04-build` | код |

## Сделано

- **Hero** — форма с живой оценкой: человек вбивает свою машину, и диапазон оффера
  ложится на бетон перед линейкой, в перспективе. Видео-готов.
- **Network** — точечная карта США, собранная из реальной геометрии штатов;
  4 шоурума Marshall Goldman с настоящими адресами, дуги перегона, автоперебор.
- **How it works** — три шага и вытащенный из стены текста аргумент:
  **чек приходит раньше эвакуатора.**

## Главное из аудита

Сайт существует, чтобы собирать заявки. Но на **30 SEO-страницах** «sell my {марка}»,
куда приходит целевой трафик, **формы нет вообще** — только одна кнопка после полутора
тысяч слов. Это чиним первым.

## ⚠️ Что нужно от клиента

- **Реальные суммы сделок.** Цены в оценке и ленте — заглушки. Показывать продавцу
  выдуманную оценку нельзя.
- **Google Reviews.** Клиент просит их в брифе, но у iBuy их нет: все ~461 отзыв
  принадлежат **Marshall Goldman** (4.3★). У самого iBuy — 0 на Trustpilot,
  ~8 на Yelp, 3 на BirdEye.
- **Kansas City.** Клиент называет его пятой локацией, но на marshallgoldman.com
  его нет. Похоже, это и есть «новый объект, который достраивается».
- **Фото без водяного знака** и **статичный план** для видео-съёмки.
- **Куда уходит заявка** — сейчас форма только логирует в консоль.
